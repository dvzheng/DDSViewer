DDS Viewer

DDS Viewer is a small program that makes you able to view DDS files and export them into PNG files.

Button Explanation

  • Open DDS - Open a DDS file.
  • Export DDS - Exports an opened DDS file into a PNG file.
  • Toggle Transparency - Toggles transparency for the opened DDS file.

Possible Questions

I keep getting "DDS file is corrupt." errors! The DDS file is either literally corrupt or a DDS format not supported by this tool.
